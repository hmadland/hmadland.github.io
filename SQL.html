SQL projects

Here are a few samples of my SQL projects. The following SQL statements use the table of information show below.

Get supplier names for suppliers who supply at least two parts without using GROUP BY.

SELECT distinct SNAME from [DataBase_1_364].[dbo].[S] WHERE sno in ( select distinct SP.SNO from [DataBase_1_364].[dbo].[SP] join [DataBase_1_364].[dbo].[SP] as SP2 on SP.SNO = SP2.SNO WHERE SP.PNO != SP2.PNO)

Using EXISTS to get supplier names for suppliers who supply at least one red part, but does not supply any green part.

SELECT distinct SNAME from [DataBase_1_364].[dbo].[S] join [DataBase_1_364].[dbo].[SP] on S.SNO = SP.SNO WHERE exists (select * from [DataBase_1_364].[dbo].[P] Where COLOR = 'Red' and SP.PNO = P.PNO and not exists ( SELECT * From [DataBase_1_364].[dbo].[P] WHERE COLOR = 'Green' ))

Get PNnames for parts have an accumulative QTY greater than 500 (using EXISTS).

SELECT PNAME from [DataBase_1_364].[dbo].[P] WHERE EXISTS ( select * from [DataBase_1_364].[dbo].[SP] GROUP BY PNO HAVING sum(QTY) > 500 and P.PNO = SP.PNO )

Get PNnames for parts have an accumulative QTY greater than 500 (using EXISTS).

SELECT PNAME from [DataBase_1_364].[dbo].[P] WHERE EXISTS ( select * from [DataBase_1_364].[dbo].[SP] GROUP BY PNO HAVING sum(QTY) > 500 and P.PNO = SP.PNO )

Using a Database with number 1-100

Write a query to list all the primes less than 100

select number from dbo.Numbers1To100 where number > 1 and number < 100 and number not in ( select distinct t1.number from dbo.Numbers1To100 as t1 join dbo.Numbers1To100 as t2 on t1.number > t2.number where t1.number < 100 and t1.number % t2.number = 0 and t2.number != 1 and t2.number != t1.number)

How many three-digit integers are increased by exactly 99 when the digits are reversed? Show the integer, the revised integer, and the difference.

select distinct t1.number, t2.number, t1.number - t2.number from dbo.ThreeDigitNumbers as t1 join dbo.ThreeDigitNumbers as t2 on t1.number != t2.number where t1.number - t2.number = 99 and t1.number > 100 and t2.number > 100 and not exists (SELECT * FROM (VALUES('%1%'), ('%2%'), ('%3%'), ('%4%'), ('%5%'), ('%6%'), ('%7%'), ('%8%'), ('%9%')) Nums(N) WHERE (t2.number like N and t1.number not like N))
